# Contributing to CrowdFlow AI

We welcome contributions from the community to help make stadiums in India smarter and safer!

## How to Contribute
1. **Report Bugs**: Use GitHub Issues to report bugs or request features.
2. **Pull Requests**: Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.
3. **Coding Standards**:
    - Use TypeScript for frontend logic.
    - Maintain 100% test coverage for new components.
    - Ensure all UI changes meet WCAG 2.1 AA standards.

## Development Workflow
1. Fork the repository.
2. Install dependencies: `cd frontend && npm install` and `cd backend && pip install -r requirements.txt`.
3. Run tests frequently: `npm test`.
